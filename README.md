-------PRUEBA MASIVIAN-------

1. Importar el proyecto en un ambiente de desarrollo JAVA.
2. Dirigirse al paquete com.masivian.prueba.
3. Dirigirse a la clase Main.java para interacctura con el proyecto, allí esta el ejercicio impuesto en la prueba (version codigo).
4. Dirigirse a la line 99 de la clase Main.java y configurar el PATH para la creacion del archivo JSON.
5. Instalar Node.js para tener acceso a su manejador de paquete NPM.
6. Despues de la instalación de Node.js, intalar el paquete json-server. 
	(npm install -g json-server)
7. Abrir la consola cmd para posteriormente ubicarse en la ruta que se configuro para crear el archibo JSON en el paso 4 y ejecutar lo siguiente:
	(json-server --watch (Nombre del archivo JSON creado en el paso 4. ejemplo: Arbol.json))
8. Luego de la ejecución, dirigirse a un navegar para posteriormente dirigirse al http://localhost:3000, alli encontrar una página en donde se podra evidenciar 
la ejecución de los dos endpoints expuestos en la prueba. (Para verificar esto ir a los Link´s Ancestro y Arbol).
